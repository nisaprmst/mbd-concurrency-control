# Database Concurrency Control

This project was created to fulfill Database Management course assignment.

## Getting Started

> With data collection, ‘the sooner the better’ is always the best answer.
>
> **Marissa Mayer**

Repo: https://github.com/nisaprmst/mbd-concurrency-control

### Prerequisites

* g++
* linux-based operating system

### Installing and Running

Steps: 

1. Clone the repository to the folder you want to host, using terminal

```
user@userComputer:~/dir git clone https://github.com/nisaprmst/mbd-concurrency-control
```

2. Run terminal on the specific folder
3. Run the following command on terminal to perform benchmarking:
```
make test
```
